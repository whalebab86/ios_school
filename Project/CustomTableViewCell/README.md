# Play Trip Diary
> 여행을 하면서 기록을 남기기 위해 만들어진 앱

# Trip Diary App Features

> 조회 : 등록한 여행일지를 최신일자를 우선으로 조회  
> 등록 : 기기의 로케이션 정보를 통하여 위치를 조회. 위치, 일자, 내용, 사진정보를 등록  
> 수정 : 위치, 일자, 내용, 사진정보를 수정
> 삭제 : 여행일지 정보를 삭제  
> 여행위치조회 : 등록된 로케이션 위치를 지도에 표시
> 세팅 : 프로필 정보를 수정
